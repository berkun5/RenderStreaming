# RenderStreaming<br/>
Create Server
--------------------
Use node.js command prompt<br/>
cd WebApp (directory)<br/>
npm install<br/>
npm run build<br/>
npm run pack

Start Server
--------------------
Use Command Prompt, powershell<br/>
.\webserver.exe -w (local port 80)<br/>
.\webserver -w -p 123 (enter a port "123" in in the example)
