# RenderStreaming
Create Server
--------------------
Use node.js command prompt
cd WebApp (directory)
npm install
npm run build
npm run pack

Start Server
--------------------
Use Command Prompt, powershell
.\webserver.exe -w (local port 80)
.\webserver -w -p 123 (enter a port "123" in in the example)
